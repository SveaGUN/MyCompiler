# MyCompiler
"やさしいコンパイラの作り方入門"の学習用

言語名 akane

予約語 if then while do return function program int const write writeln writesp
記号　{} () スペース + - * /  比較 = < > <> <= >= 代入 :=  , ;

コマンドラインのオプション
-printLine -p ソースコード行を出力
-genList -g リスティングファイルの生成
-execute -e コンパイル後に仮想マシンで実行
-dispCode -d 仮想マシンで実行しているコードの出力
-verbose -v 仮想マシンで詳細な情報の出力
